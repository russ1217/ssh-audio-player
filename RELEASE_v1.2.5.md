# SSH Audio Player v1.2.5

## 🎉 新功能：全屏播放模式

本次更新带来了全新的**全屏播放模式**，为用户提供沉浸式的音频播放体验！

### ✨ 主要特性

#### 📱 全屏播放模式
- **横屏显示**：自动切换到横屏，提供更宽的视野
- **秒表式计时器**：80px 超大字体显示当前播放时间（MM:SS 或 HH:MM:SS）
- **进度条控制**：可拖动调节播放位置，实时反馈
- **时长显示**：右侧显示"当前时间 / 总时长"
- **防锁屏功能**：使用 wakelock_plus 保持屏幕常亮
- **沉浸式体验**：隐藏状态栏和导航栏
- **完整生命周期管理**：前后台切换、锁屏解锁等场景完美处理

#### 🎨 UI 优化
- **两行控制布局**：重新设计底部播放控制栏
  - 第一行：主要控制（上一曲、快退、播放/暂停、快进、下一曲）
  - 第二行：辅助功能（播放模式、停止、**全屏按钮**）
- **全屏按钮突出**：更大的图标（28px），主题色高亮

### 🔧 技术改进

- 新增依赖：`wakelock_plus ^1.2.8`
- 使用 `SystemChrome` 精确控制屏幕方向
- StreamBuilder 实现 UI 实时更新
- 完善的资源管理和清理机制

### 💡 使用提示

**如何进入全屏模式：**
1. 播放任意音频文件
2. 在底部控制栏找到第二行最右侧的 **⛶ 全屏按钮**
3. 点击进入全屏模式
4. 点击"退出全屏"返回正常模式

**适用场景：**
- 🏃 运动健身时监控音乐进度
- 🧘 冥想练习时查看时间
- 🎵 DJ 混音时精确控制
- 📚 学习时听长音频课程
- 🎤 KTV 唱歌时跟踪进度

### 🐛 已知问题

- 某些定制 Android 系统可能会覆盖防锁屏设置
- 低电量模式下系统可能忽略防锁屏请求
- 系统来电或闹钟可能暂时中断全屏体验
