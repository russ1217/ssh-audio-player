# 全屏播放功能快速开始

## 🚀 30秒快速体验

### 步骤 1：启动应用
```bash
flutter run
```

### 步骤 2：播放音频
1. 选择本地文件或连接 SSH
2. 点击任意音频文件开始播放

### 步骤 3：进入全屏
在底部控制栏，点击最右侧的 **⛶ 全屏按钮**

### 步骤 4：享受体验
- 📱 横屏显示
- ⏱️ 超大字体计时器
- 📊 可拖动进度条
- 🔒 屏幕不会锁定
- 🎮 基本播放控制

### 步骤 5：退出全屏
点击底部的 **"退出全屏"** 按钮

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🔄 横屏显示 | 自动切换，退出时恢复 |
| ⏱️ 秒表计时 | 80px 大字体，MM:SS 或 HH:MM:SS |
| 📊 进度控制 | 可拖动，实时更新 |
| 🔒 防锁屏 | wakelock_plus 保持常亮 |
| 🎮 播放控制 | 快退、播放/暂停、快进 |
| 🚪 一键退出 | 点击按钮返回正常模式 |

## 💡 使用场景

### ✅ 适合的场景
- 🏃 运动健身时监控音乐进度
- 🧘 冥想练习时查看时间
- 🎵 DJ 混音时精确控制
- 📚 学习时听长音频课程
- 🎤 KTV 唱歌时跟踪进度

### ❌ 不适合的场景
- 🔋 电池电量低时（会增加耗电）
- 🚌 移动环境中（横屏不便观看）
- 📞 需要频繁接听电话时

## ⚙️ 技术实现

### 关键依赖
```yaml
wakelock_plus: ^1.2.8  # 防止锁屏
```

### 核心代码位置
- **屏幕组件**：`lib/screens/fullscreen_player_screen.dart`
- **入口按钮**：`lib/widgets/bottom_player_bar.dart`
- **状态管理**：`lib/providers/app_provider.dart`

## 🔧 自定义修改

### 调整字体大小
编辑 `fullscreen_player_screen.dart`：
```dart
fontSize: 80,  // 改为你想要的大小，如 60 或 100
```

### 更改颜色主题
```dart
color: Colors.white,  // 改为其他颜色
activeTrackColor: Colors.blueAccent,  // 进度条颜色
```

### 修改时间格式
编辑 `_formatTimeAsStopwatch` 方法：
```dart
// 始终显示小时
return '${_twoDigits(hours)}:${_twoDigits(minutes)}:${_twoDigits(seconds)}';
```

### 添加更多控制按钮
在控制按钮区域添加：
```dart
IconButton(
  icon: const Icon(Icons.skip_previous, size: 40, color: Colors.white),
  onPressed: () => provider.playPreviousInPlaylist(),
),
```

## 🐛 常见问题

### Q: 为什么屏幕还是会锁定？
**A:** 
1. 检查系统电池优化设置
2. 确认应用有"保持唤醒"权限
3. 某些系统（如 MIUI）需要在安全中心手动授权

### Q: 如何在全屏时切换到下一首？
**A:** 当前版本暂不支持，需要退出全屏后操作。未来版本会添加。

### Q: 全屏模式下能看歌词吗？
**A:** 当前版本不支持歌词显示，这是未来的改进计划之一。

### Q: 可以竖屏全屏吗？
**A:** 当前设计为横屏以获得更好的观看体验。如需竖屏，可修改代码中的方向设置。
