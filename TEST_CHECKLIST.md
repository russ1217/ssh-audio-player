# 播放模式功能测试清单

## 📋 测试前准备

1. 确保应用已编译成功：`flutter build apk --debug`
2. 在Android设备或模拟器上安装应用
3. 准备一个包含至少5首歌曲的播放列表

## ✅ 功能测试

### 1. 正常播放模式（默认）
- [ ] 启动应用，确认默认是正常播放模式
- [ ] 播放列表中的歌曲按顺序播放
- [ ] 播放到最后一首后自动停止
- [ ] 可以使用上一曲/下一曲按钮
- [ ] 暂停后可以正常恢复

### 2. 列表循环模式
- [ ] 点击播放模式按钮切换到列表循环（图标变为🔁白色）
- [ ] 播放到最后一首后自动回到第一首继续
- [ ] 连续循环3次验证稳定性
- [ ] 暂停/恢复功能正常
- [ ] 上一曲/下一曲功能正常

### 3. 单曲循环模式
- [ ] 切换到单曲循环模式（图标变为🔂白色）
- [ ] 当前歌曲播放完成后自动重新开始
- [ ] 验证重复播放3次以上
- [ ] 暂停后恢复仍保持单曲循环
- [ ] seek操作后仍能单曲循环

### 4. 随机播放模式
- [ ] 切换到随机播放模式（图标变为🔀白色）
- [ ] 点击下一曲，验证是随机选择的歌曲
- [ ] 点击上一曲，能正确回溯到前一首
- [ ] 播放完所有歌曲后重新生成随机顺序
- [ ] 验证随机性（多次切换模式观察不同顺序）

## 🔒 兼容性测试

### 5. 断网重播
- [ ] 在正常模式下断开网络，验证暂停
- [ ] 恢复网络，验证能从断点续播
- [ ] 在列表循环模式下重复上述测试
- [ ] 在单曲循环模式下重复上述测试
- [ ] 在随机播放模式下重复上述测试
- [ ] 确认恢复后播放模式保持不变

### 6. 暂停/播放
- [ ] 在各种模式下测试暂停功能
- [ ] 暂停后等待1分钟再恢复
- [ ] 验证恢复位置准确（误差<1秒）
- [ ] 确认播放模式未改变

### 7. 后台播放
- [ ] 在各种模式下切换到后台
- [ ] 验证后台继续播放
- [ ] 通知栏显示正确的播放模式
- [ ] 通过通知栏控制播放（暂停/下一曲等）
- [ ] 切回前台，状态一致

### 8. 播放列表操作
- [ ] 添加新歌曲到播放列表
- [ ] 删除正在播放的歌曲
- [ ] 清空播放列表
- [ ] 验证随机模式在清空后重置
- [ ] 重新添加歌曲后启用随机模式

### 9. 睡眠定时器
- [ ] 设置睡眠定时器
- [ ] 在各种模式下验证定时器正常工作
- [ ] 定时器到期后正确停止播放
- [ ] 取消定时器功能正常

### 10. 通知栏控制
- [ ] 下拉通知栏查看媒体控制
- [ ] 通过通知栏切换歌曲
- [ ] 验证通知栏显示的歌曲信息正确
- [ ] 锁屏界面控制功能正常
