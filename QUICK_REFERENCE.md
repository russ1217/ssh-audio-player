# 快速参考卡

## 5 分钟上手

### 1️⃣ 添加服务器（30 秒）
```
设置 → SSH 服务器配置 → + 按钮
填写：名称、主机、用户名、密码
保存
```

### 2️⃣ 连接服务器（10 秒）
```
在 SSH 列表找到你的服务器
点击左边的 → 登录图标
等待连接成功
```

### 3️⃣ 播放音乐（10 秒）
```
切换到"文件"标签
点击音频文件
开始播放！
```

### 4️⃣ 添加到播放列表（5 秒）
```
长按文件
选择"添加到播放列表"
```

### 5️⃣ 设置定时关闭（5 秒）
```
设置 → 定时关闭
选择时间（如 30分钟）
完成！
```

## 常用操作快捷键

### 播放控制
| 按钮 | 功能 | 说明 |
|------|------|------|
| ⏮️ | 上一曲 | 播放上一首 |
| ⏪ | 快退 | 后退 10 秒 |
| ▶️ | 播放 | 开始/继续播放 |
| ⏸️ | 暂停 | 暂停播放 |
| ⏩ | 快进 | 前进 10 秒 |
| ⏭️ | 下一曲 | 播放下一首 |
| ⏹️ | 停止 | 停止播放 |

### 进度条
```
拖拽圆点 → 跳转到任意位置
左边时间 → 已播放时间
右边时间 → 总时长
```

## 常见问题速查

### ❌ 连接失败
```
✓ 检查服务器地址
✓ 检查用户名密码
✓ 检查网络连接
✓ 确认 SSH 服务运行中
```

### 🐌 播放卡顿
```
✓ 等待缓冲完成
✓ 改善网络连接
✓ 使用较小的文件
```

### 📦 添加整个目录
```
在文件浏览器
点击右上角 📝 图标
自动添加所有媒体文件
```

### 💾 保存播放列表
```
在播放列表页面
点击保存图标
输入名称 → 确定
```

## 提示与技巧

### 💡 技巧 1: 顺序播放专辑
```
1. 进入专辑文件夹
2. 点击右上角添加目录按钮
3. 自动按名称排序播放
```

### 💡 技巧 2: 睡前模式
```
1. 添加喜欢的音乐
2. 设置定时 30 分钟
3. 锁屏
4. 安心入睡
```

### 💡 技巧 3: 视频听音频
```
直接点击视频文件
自动提取音频播放
```
