# SSH 音频播放器 - 使用指南

## 快速开始

### 第一次使用

1. **启动应用** - 打开应用后你会看到三个标签页：文件、播放列表、设置

2. **配置 SSH 服务器**
   - 切换到"设置"标签页
   - 点击"SSH 服务器配置"
   - 点击右下角的 "+" 按钮添加新服务器
   - 填写服务器信息：
     ```
     名称：我的服务器（随便取）
     主机地址：192.168.1.100 或 example.com
     端口：22（默认）
     用户名：your_username
     密码：your_password
     初始路径：/home/user/music（你想从哪里开始播放）
     ```
   - 点击"保存"

3. **连接服务器**
   - 在 SSH 配置列表中找到刚才添加的服务器
   - 点击左边的"登录"图标（箭头图标）
   - 等待连接成功
   - 连接成功后会自动进入初始目录

4. **浏览和播放**
   - 切换到"文件"标签页
   - 点击文件夹图标进入子目录
   - 点击音频文件（蓝色音乐图标）开始播放
   - 底部会出现播放控制栏

### 播放控制

底部播放控制栏从左到右：

```
⏮️  ⏪  ▶️/⏸️  ⏩  ⏭️  ⏹️
上一曲 快退  播放/暂停  快进  下一曲  停止
```

**进度条**：可以直接拖拽跳转到任意位置

**快进/快退**：每次跳转 10 秒

### 播放列表

**添加文件到播放列表**：
- 方法1：长按文件 → 选择"添加到播放列表"
- 方法2：在文件浏览器右上角点击"添加目录"按钮，将当前目录所有媒体文件添加到播放列表

**管理播放列表**：
- 切换到"播放列表"标签页
- 查看当前播放队列
- 点击文件直接播放
- 点击删除图标（垃圾桶）移除单个文件
- 点击"保存"图标保存播放列表到数据库（下次启动还在）
- 点击"清空"图标清空整个播放列表

### 定时关闭

**设置定时**：
1. 切换到"设置"标签页
2. 点击"定时关闭"
3. 选择以下方式之一：
   - **按时间**：选择 15分钟、30分钟、1小时、2小时、3小时、6小时
   - **按文件数**：点击"播放 N 个文件后关闭"，输入数量
4. 定时开始后会自动倒计时

**取消定时**：
1. 再次进入"定时关闭"
2. 点击"取消定时"

### 后台播放

应用切换到后台后会自动继续播放：
- 按 Home 键返回桌面
- 切换到其他应用
- 锁屏

**注意**：目前通知栏控制还在开发中，暂时需要在应用内控制播放。

## 常见问题

### Q: 连接失败怎么办？

**检查清单**：
1. 服务器地址和端口是否正确
2. 用户名和密码是否正确
3. 服务器 SSH 服务是否运行
4. 网络连接是否正常
5. 防火墙是否允许 SSH 端口

### Q: 播放卡顿怎么办？

**可能原因**：
1. 网络速度慢 - 文件需要从服务器下载
2. 文件太大 - 大文件需要更长的缓冲时间

**解决方法**：
- 等待缓冲完成再开始播放
- 尝试降低音频质量或使用较小的文件
- 改善网络连接

### Q: 支持哪些文件格式？

**音频格式**：
- MP3 (.mp3)
- WAV (.wav)
- FLAC (.flac)
- AAC (.aac)
- OGG (.ogg)
- M4A (.m4a)
- WMA (.wma)
- OPUS (.opus)
- AIFF (.aiff)

**视频格式（只播放音频）**：
- MP4 (.mp4)
- FLV (.flv)
- MKV (.mkv)
- AVI (.avi)
- MOV (.mov)
- WMV (.wmv)
- WEBM (.webm)
- M4V (.m4v)

### Q: 播放列表会保存吗？

**会！** 有两种保存方式：

1. **自动保存**：当前播放列表在应用运行期间一直存在
2. **手动保存**：点击"保存"图标，输入名称，保存到数据库
   - 保存后即使关闭应用也不会丢失
   - 未来版本会支持加载已保存的播放列表

### Q: 如何播放视频文件的音频？

直接点击视频文件即可！应用会自动提取音频并播放，不会显示视频画面。

### Q: 定时关闭准确吗？

- **按时间**：基本准确，基于系统计时器
- **按文件数**：在每个文件播放完成后计数

## 高级技巧

### 技巧 1: 快速添加整个专辑

1. 进入专辑文件夹
2. 点击右上角"添加目录"按钮
3. 所有音频文件会自动按名称排序添加到播放列表
4. 从第一首开始顺序播放

1. 添加喜欢的音乐到播放列表
2. 设置定时关闭（比如 30 分钟）
3. 开始播放后锁定屏幕
4. 安心入睡，音乐自动停止

### 技巧 3: 播客连续播放

1. 添加播客文件夹到播放列表
2. 文件会按名称顺序播放
3. 使用快进/快退定位到想听的部分

## 技术细节

### 文件播放原理

1. 通过 SSH 从服务器下载文件到本地临时目录
2. 使用 just_audio 播放本地临时文件
3. 播放完成后临时文件保留（下次播放无需重新下载）

### 数据存储

应用使用 SQLite 数据库存储：
- SSH 服务器配置
- 保存的播放列表
- 播放历史记录（未来功能）

数据库位置：`/data/data/com.audioplayer.ssh_audio_player/databases/ssh_audio_player.db`

### 权限说明

应用申请的权限：
- **INTERNET** - 连接 SSH 服务器（必须）
- **FOREGROUND_SERVICE** - 后台播放（必须）
- **WAKE_LOCK** - 防止播放时休眠（必须）
- **存储权限** - 缓存音频文件（可选）
